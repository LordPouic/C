Via une boucle while puis une boucle for afficher tout les chiffres entiers de 100 Ã  1


```
int i =100;
while (i>=0)
{
    printf("%d",i);
    i--;
}
```

```
for (int i=100;i>=0;i--)
{
    printf("%d",i);
}
```

Ecrire un programme qui calcule la somme des 10 premiers chiffres entiers


```
int s =0;
for(int i=0;i<10;i++)
{
    s += i;
}
printf("%d",s);
```
